## Columns多列布局

**column-width | column-count; ** 

​	`column-width:`每列的宽度

​	`column-count:`显示的列数



`column-gap: `列间距宽，默认大小由 font-size 决定



**column-rule:** width style color 列边框样式 (不占宽度)

​	`column-rule-width  `	

​	`column-rule-style `

​	`column-style-color  ` 



`column-span: none | all`    标题横跨所有列
