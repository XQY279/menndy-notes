## 基本命令

#### `cd`  ：切换目录

**cd命令**用来切换工作目录至[dirname](http://man.linuxde.net/dirname)。 其中dirName表示法可为绝对路径或相对路径。若目录名称省略，则变换至使用者的home directory(也就是刚[login](http://man.linuxde.net/login)时所在的目录)。另外，`~`也表示为home directory的意思，`.`则是表示目前所在的目录，`..`则表示目前目录位置的上一层目录。

| 命令              | 描述                                                         |
| :---------------- | ------------------------------------------------------------ |
| cd -p             | 如果要切换到的目标目录是一个符号连接，直接切换到符号连接指向的目标目录 |
| cd -L             | 如果要切换的目标目录是一个符号的连接，直接切换到字符连接名代表的目录，而非符号连接所指向的目标目录。 |
| cd -              | 返回进入此目录之前所在的目录；                               |
| cd                | 进入用户主目录；                                             |
| cd ~              | 进入用户主目录；                                             |
| cd ..             | 返回上级目录（若当前目录为“/“，则执行完后还在“/"；".."为上级目录的意思）； |
| cd ../..          | 返回上两级目录；                                             |
| cd !$             | 把上个命令的参数作为cd参数使用。                             |
| cd `<filename>  ` | 进入某一个目录                                               |
| cd /              | 进入到根目录                                                 |





#### `ls`   : 列出当前文件夹下的内容（可以是文件或者文件夹）

| 参数 | 描述                                                         |
| ---- | ------------------------------------------------------------ |
| -a   | 显示所有档案及目录（ls内定将档案名或目录名称为“.”的视为影藏，不会列出） |
| -A   | 显示除影藏文件“.”和“..”以外的所有文件列表；                  |
| -l   | 以长格式显示目录下的内容列表。输出的信息从左到右依为：`访问权限` `当前目录之内存在的目录的数量` `当前这个文件属于哪个用户` `当前这个文件属于哪个用户组` `当前目录或文件的大小` `后三个为创建文件的时间` `目录或文件的名称` |





#### `mkdir` 创建目录

| 命令               | 描述                     |
| ------------------ | ------------------------ |
| mkdir `<fileName>` | 在当前目录下创建一个目录 |



##### `touch <filename>` 新建文件 

##### `vi <filename>` 进入文件	

##### ` :wq` 退出当前文件

##### `cat <filename> 查看当前文件`



#### `cp ` 复制一个文件

| 参数     | 描述                                                         |
| -------- | ------------------------------------------------------------ |
| 源文件   | 制定源文件列表。默认情况下，cp命令不能复制目录，如果要复制目录，则必须使用`-R`选项； |
| 目标文件 | 指定目标文件。当“源文件”为多个文件时，要求“目标文件”为指定的目录。 |



| 命令    | 描述                                               |
| ------- | -------------------------------------------------- |
| cp      | 复制一个文件                                       |
| cp -R/r | 递归处理，将指定目录下的所有文件与子目录一并处理； |
|         |                                                    |



#### `pwd` 显示当前目录的全部路径



#### `rm` 删除文件

| 命令                   | 描述         |
| ---------------------- | ------------ |
| rm  <要删除的文件名>   | 删除一个文件 |
| rm -R <要删除的目录名> | 删除一个目录 |



#### `shutdown -h now`   立刻关机

#### `shutdown -r now/reboot`   立刻重启计算机
